# Monitoreo de Seguridad Personal en Redes Wi-Fi

## Descripción
Aplicación móvil que permite detectar amenazas en redes Wi-Fi (como ARP Spoofing),
mostrando alertas en lenguaje simple para usuarios sin conocimientos técnicos.

## Stack tecnológico
- Frontend: React Native (JavaScript/TypeScript)
- Backend: Python con FastAPI
- Base de datos: SQLite

  - backend/app: api, services, models, db, core
